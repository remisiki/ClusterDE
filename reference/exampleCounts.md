# A count matrix of 100 genes from the A549 cell line from Tian et al., Nature Methods, 2019.

A count matrix of 100 genes from the A549 cell line from Tian et al.,
Nature Methods, 2019.

## Usage

``` r
data("exampleCounts")
```

A dataset with 100 rows (genes) and 1256 cols (cells)
